# 🐍 Jogo da Cobrinha

Um jogo da cobrinha desenvolvido com **HTML**, **CSS** e **JavaScript**, recriando o clássico Snake de forma simples e interativa. O projeto utiliza manipulação do DOM, lógica de movimentação, detecção de colisões e geração aleatória de alimentos, proporcionando uma experiência divertida e responsiva.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

## Funcionalidades

* Movimentação da cobrinha pelo teclado.
* Geração aleatória de alimentos.
* Aumento do tamanho da cobrinha ao comer.
* Sistema de pontuação.
* Detecção de colisão com as bordas e com o próprio corpo.
* Reinício da partida após o fim do jogo.

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, lógica de programação e manipulação de eventos utilizando JavaScript puro.
